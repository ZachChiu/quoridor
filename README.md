# 牆壁圍棋 Wall Go

靈感來自於 Netflix 熱門影集《魔鬼的計謀 2》的策略與心理戰元素。玩家需要在棋盤上佈局牆壁，運用智慧與謀略，阻擋對手同時佔領更多領地。

## 遊戲網址

[https://quoridorgame.com/](https://quoridorgame.com/)

## 技術棧

- **前端框架**：Next.js 15、React 19
- **UI**：Tailwind CSS
- **語言**：TypeScript
- **工具庫**：Lodash
- **圖標**：React Icons
- **分析**：Google Analytics
- **SEO**：Next Sitemap
